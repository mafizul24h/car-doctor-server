## Tech Stack

- **Frontend**: HTML, CSS, Tailwind CSS, Daisyui, React JS, Context API, Firebase
- **Backend**: Node.js, Express.js, MongoDB
- **Authentication & Security**: Firebase User Authentication, JWT for enhanced security
